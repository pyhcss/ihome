# 接口说明

1 图片验证码
url:    /api/imagecode?pre=&cur=
method: GET
描述： 生成图片验证码
传入参数   类型           说明            是否允许为空
 pre     string     上一张验证码id           是
 cur     string   需要获取的验证码id          否
返回值:图片二进制数据

2 手机验证码
url:    /api/telcode
method: POST
描述：　校验图片验证码并发送短信验证码
    传入参数    类型/json        说明        是否允许为空
    mobile      string        手机号           否
   imagecode    string     图片验证码文本　      否
 imagecode_id   string     图片验证码id         否

    　返回值    类型/json        说明
    errcode     string      　错误编号
    errmsg      string      错误信息描述

3 注册接口
url:    /api/register
method: POST
描述： 接收客户端参数并生成session返回给客户端
    传入参数    类型/json          说明          是否允许为空
    mobile      string          手机号              否
    phonecode    string        短信验证码            否
    password      string         密码               否

    返回值      类型/json        说明
    errcode     string      　错误编号
    errmsg      string      错误信息描述

area           string         所属区域                 否
address        string          地址                   否
count          string         房间个数                 否
acreage        string         房屋面积                 否
type           string         房屋类型                 否
num            string         容纳人数                 否
beds           string         配床描述                 否
deposit        string         押金数                   否
max_day        string        最多入住天数               否
min_day        string        最少入住天数               否

 返回值     　类型/json      　   说明
errcode       string      　   错误编号
errmsg        string         错误信息描述
house_id      string           房屋id

17 房屋列表页接口
url:    /api/houselist
method: GET
描述: 房屋列表页获取数据 分页等
传入参数        类型/json        说明           是否允许为空
sd             string         开始时间              是
ed             string         结束时间              是
aid            string          区域id               是
sk             string      new时间倒序 booking销量   是
                        price-inc正序 price-des倒序
p              string        页面编号

 返回值     　类型/json      　   说明
errcode       string      　   错误编号
errmsg        string         错误信息描述
total_page    string          总页面数
data          json列表    房屋id house_id 图片image_url 房东头像avatar
                         单价price 标题title 房间数room_count 销量order_count
                         地址address
